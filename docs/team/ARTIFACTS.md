## Coordination Artifacts
- coordination_note

## Artifact Status Lifecycle
- draft
- submitted
- approved
- rejected

## Review Metadata
- reviewed_by must be different from created_by
- approved / rejected require submitted_at first
- review_decision should be APPROVE / REJECT / REQUEST_CHANGES as applicable
- reviewer should receive only the minimum context needed to review the submitted change
